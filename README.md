# EzSS - Modern and Minimal Styling

EzSS Styling is a collection of premade CSS styles that can be used to develop fast and easy prototypes that implements Modern and Minimal design.

## Basics
- Div
- Headings
- Fonts

## Components
- Navigation Bars
- Buttons
- Inputs
- Alert Notifications
